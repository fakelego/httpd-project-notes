# Linux 基础操作笔记

## 文件操作
- pwd: 显示当前目录
- ls: 列出文件
- cd: 切换目录
- mkdir: 创建目录
- touch: 创建文件
- cp: 复制
- mv: 移动/重命名
- rm: 删除

## 重要目录
- /home: 用户主目录
- /usr: 用户程序
- /etc: 配置文件
- /var: 可变数据
- /tmp: 临时文件

## 权限管理
- r (read): 读权限
- w (write): 写权限
- x (execute): 执行权限

chmod +x: 添加执行权限
chmod -x: 移除执行权限
chmod 755: 数字方式设置权限
